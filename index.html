Deliverables

iPhone

  • Program: Blender
  • Start date: date
  • Finish Date: date
  • Detail

    In my first project, I made a 3D model of an iPhone in blender. I used a blueprint as a reference and created the shape of the iphone using tools such as bevel, insert faces, scaling, and transformations. To create the texture and color of the phone, I created and applied appropriate materials such as black glass, plastic, and metal to each parts of the phone. The phone screen was done by UV editing.

Speaker

  • Program: Blender
  • Start date: date
  • Finish Date: date
  • Detail

    In my second project, I made a 3D model of a speaker in blender. I used 2 blueprints each showing the back and front of the speaker as a reference and created the main body of the speaker and the additional elements using different tools. While doing so, I learned to use new tools such as extrude, knife tool, and loop cut. I used UV editing for the Yamaha logo and the back part of the speaker.
